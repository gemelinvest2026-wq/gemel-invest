# GEMEL INVEST Mobile

שלד אפליקציה ראשוני ל־Capacitor עבור GEMEL INVEST.

## מה יש בפרויקט
- מסך פתיחה פרימיום
- הגדרות Capacitor
- בסיס מוכן ל־Android ו־iOS
- תיקיית `www` לקבצי המערכת

## פקודות התחלה
```bash
npm install
npx cap add android
npx cap sync
npx cap open android
```

## בהמשך
כאן נחבר את קבצי המערכת הראשיים שלך במקום מסך הפתיחה.
